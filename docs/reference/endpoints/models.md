# Models Endpoints

Total operations: 6

## GET `/v1/models`

### Purpose

Retrieve models.

List all available models (OpenAI-compatible endpoint)

### Authentication

Public (no authentication required).

### Parameters

| Name | In | Required | Type | Description |
|---|---|---|---|---|
| `scope` | query | no | string | runnable=only providers with active adapters; all/discovered=full discovered inventory |
| `endpoint` | query | no | string | Optional endpoint compatibility filter. Canonical: chat_completions, responses, completions, embeddings, images, videos, audio_speech, audio_transcriptions, realtime. Aliases accepted (e.g. chat, tts, stt). |

### Request Body

No JSON request body is required.

### Responses

| Status | Description |
|---|---|
| `200` | List of models |
| `400` | #/components/responses/BadRequest |
| `401` | #/components/responses/Unauthorized |
| `403` | #/components/responses/Forbidden |
| `404` | #/components/responses/NotFound |
| `409` | #/components/responses/Conflict |
| `422` | #/components/responses/UnprocessableEntity |
| `429` | #/components/responses/TooManyRequests |
| `500` | #/components/responses/InternalServerError |

### Error Handling

Client errors generally follow 4xx contracts (`BadRequest`, `Unauthorized`, `Forbidden`, `NotFound`, `Conflict`, `UnprocessableEntity`, `TooManyRequests`). Server failures return `500`.

### Rate Limits

Subject to tenant-level quota and platform-level rate-limit policies.

### Observability

Propagate and log `X-Request-Id` and `X-Correlation-Id` for traceability, debugging, and audit workflows.

### Examples

```bash
curl -X GET "https://api.ailin.one/v1/models" \
  -H "Authorization: Bearer $AILIN_TOKEN" \
  -H "X-API-Key: $AILIN_API_KEY"
```

```ts
const response = await fetch("https://api.ailin.one/v1/models", {
  method: "GET",
  headers: {
    Authorization: `Bearer ${process.env.AILIN_TOKEN}`,
    "X-API-Key": process.env.AILIN_API_KEY || "",
  },
});
const data = await response.json();
```

```python
import os
import requests

response = requests.request(
    "GET",
    "https://api.ailin.one/v1/models",
    headers={
        "Authorization": f"Bearer {os.environ.get('AILIN_TOKEN', '')}",
        "X-API-Key": os.environ.get("AILIN_API_KEY", ""),
    },
)
print(response.status_code)
print(response.text)
```

## GET `/v1/models/{id}`

### Purpose

Retrieve models id.

Get specific model details

Provider/model IDs are accepted with raw "/" or encoded "%2F". Prefer encoded "%2F" for maximum gateway/client compatibility.

### Authentication

Public (no authentication required).

### Parameters

| Name | In | Required | Type | Description |
|---|---|---|---|---|
| `id` | path | yes | string | Model ID. Supports "provider/model" and encoded "provider%2Fmodel". Only one provider/model separator is supported. |

### Request Body

No JSON request body is required.

### Responses

| Status | Description |
|---|---|
| `200` | Model details |
| `400` | #/components/responses/BadRequest |
| `401` | #/components/responses/Unauthorized |
| `403` | #/components/responses/Forbidden |
| `404` | #/components/responses/NotFound |
| `409` | #/components/responses/Conflict |
| `422` | #/components/responses/UnprocessableEntity |
| `429` | #/components/responses/TooManyRequests |
| `500` | #/components/responses/InternalServerError |

### Error Handling

Client errors generally follow 4xx contracts (`BadRequest`, `Unauthorized`, `Forbidden`, `NotFound`, `Conflict`, `UnprocessableEntity`, `TooManyRequests`). Server failures return `500`.

### Rate Limits

Subject to tenant-level quota and platform-level rate-limit policies.

### Observability

Propagate and log `X-Request-Id` and `X-Correlation-Id` for traceability, debugging, and audit workflows.

### Examples

```bash
curl -X GET "https://api.ailin.one/v1/models/ai21%2Fjamba-large-1.7" \
  -H "Authorization: Bearer $AILIN_TOKEN" \
  -H "X-API-Key: $AILIN_API_KEY"
```

```ts
const modelId = "ai21/jamba-large-1.7";
const encodedId = encodeURIComponent(modelId);
const response = await fetch(`https://api.ailin.one/v1/models/${encodedId}`, {
  method: "GET",
  headers: {
    Authorization: `Bearer ${process.env.AILIN_TOKEN}`,
    "X-API-Key": process.env.AILIN_API_KEY || "",
  },
});
const data = await response.json();
```

```python
import os
import requests
from urllib.parse import quote

model_id = "ai21/jamba-large-1.7"
encoded_id = quote(model_id, safe="")

response = requests.request(
    "GET",
    f"https://api.ailin.one/v1/models/{encoded_id}",
    headers={
        "Authorization": f"Bearer {os.environ.get('AILIN_TOKEN', '')}",
        "X-API-Key": os.environ.get("AILIN_API_KEY", ""),
    },
)
print(response.status_code)
print(response.text)
```
